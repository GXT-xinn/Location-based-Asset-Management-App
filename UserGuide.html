Overview

  • Item Name : Condition Assessment App
  • Item Version : v 1.0
  • Author : Xintong Guan
  • Module : CEGE0043

This user guide is designed for the Condition Assessment App, which you will be able to:

  1. View the latest condition of all assets created by you based on the colour;
  2. Submit condition report for any assets created by you;
  3. Know their rank in the database based on the reports submitted;
  4. View only 5 assets closest to the you current location;
  5. View only 5 assets which associated reports were created by you most recently;
  6. View all assets in database that have not been rated in last 3 days.

As this server is inside the UCL system, you will need to be on the UCL VPN before you can connect. See here: https://www.ucl.ac.uk/isd/services/get-connected/ucl-virtual-private-network-vpn

Getting Started

To do the Condition Assessment, you have to perform on a small/medium screen devices, including, but not limited to, smartphones or a minimized (50%) the browser window.

Load the Website

In a browser (e.g. edge, chrome) type the following URL:

https://cege0043-2022-45.cs.ucl.ac.uk/app/bootStrap.html

Then, you should see a screen similar to the following:

Noticeably, markers are colour-coded based on its condition. Each colour reflect on a specific condition specified below:

• Element is in very good condition
• Some aesthetic defects, needs minor repair
• Functional degradation of some parts, needs maintenance
• Not working and maintenance must be done as soon as reasonably possible
• Not working and needs immediate, urgent maintenance
• Unknown (Unrated)

Rate An Asset

As the condition form of the closest asset has popped up, you can see the NAME and ID of the asset to double confirm that you are rating the correct asset.

However, if you want to rate other asset rather than the closest one, you can close the condition form by PRESS/CLICK on the “x” at the top-right corner of the form. By PRESSING on markers, you should be able to see another condition form containing associated asset’s information.

  • Step 1 - Select the condition description in form, which describes your selected asset the most accurate.

    • **Make sure you made your condition selection**

  • Step 3 - The web page will reload once you confirmed the all alert messages. The marker with colour will be updated corresponding to the asset’s latest condition description. You can re-check colour with this user guide (see section 1.1.1)

Add/Remove Layers

5 Closest Assets

• Add Layer – 5 closest assets
This navigation button allows you to find the 5 assets closest to the your current location, added by any user. Map will zoom to focus on the 5 markers. You should see a screen similar to the one below.



• Remove Layer – 5 closest assets
Remove the 5 closest assets from map and return back to the original map as you saw when you open the web-link.

Last 5 Reports

• Add Layer – last 5 reports, colour coded
This navigation button allows you to find the last 5 reports created by the you. The result will show as colour-coded points on the map, based on their condition values. Map will zoom to focus on the 5 markers. You should see a screen similar to the one below.



• Remove Layer – last 5 reports, colour coded
Remove the 5 color-coded markers from map and return back to the original map as you saw when you open the web-link.

Not Rated in 3 Days

• Add Layer – not rated in the last 3 days
This navigation button allows you to find the assets that user hasn't rated in the last 3 days. Map will zoom to contain all markers in database that are not rated in last three days. You should see a screen similar to the one below.



• Remove Layer – not rated in the last 3 days
Remove the unrated markers from map and return back to the original map as you saw when you open the web-link.
